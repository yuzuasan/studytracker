# ■ 使用技術一覧（StudyTracker）

---

## ■ 1. バックエンド

| 項目      | 技術          |
| ------- | ----------- |
| 言語      | Java        |
| フレームワーク | Spring Boot |

---

## ■ 2. フレームワーク / ライブラリ

| 分類       | 技術                                  | 用途         |
| -------- | ----------------------------------- | ---------- |
| Web      | Spring Web                          | REST API構築 |
| ORM      | Spring Data JPA                     | DBアクセス     |
| セキュリティ   | Spring Security                     | 認証・認可      |
| 認証方式     | JWT（JSON Web Token）                 | トークン認証     |
| バリデーション  | Bean Validation（Jakarta Validation） | 入力チェック     |
| パスワード暗号化 | BCrypt                              | パスワードハッシュ化 |
| API確認    | Swagger（springdoc-openapi）          | APIドキュメント  |
| 開発補助     | Lombok                              | ボイラープレート削減 |

---

## ■ 3. データベース

| 項目       | 技術                   |
| -------- | -------------------- |
| DB       | H2 Database          |
| 接続       | JDBC                 |
| マイグレーション | 未使用（必要に応じてFlyway等検討） |

---

## ■ 4. 認証・セキュリティ

| 項目       | 内容             |
| -------- | -------------- |
| 認証方式     | JWT（ステートレス）    |
| トークン署名   | HS256          |
| トークン有効期限 | 1時間            |
| ログアウト    | クライアント側でトークン破棄 |
| パスワード    | BCryptでハッシュ化   |
| 認可       | userIdベース制御    |
| フィルタ     | JWT認証フィルタ      |
